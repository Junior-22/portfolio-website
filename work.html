Pokedex-App

I built a small Pokedex web app with Javascript, HTML, CSS, jQuery, and the Bootstrap framework. The code has been formatted according to ESLint rules. Users can search through a collection of Pokemons that is loaded from an external API. Images and information about a specific Pokemon is displayed in a modal view by clicking on the Pokemons name.

Server-side: movie_api

API built for a movie application which allows users to access a database to receive information on movies. Also, the user can get background information on the movies. The user can sign up, update personal information and save/delete favorite movies to/from his/her personal profile. It’s written in Node.js and it's packages are stored in a database built using MongoDB.
